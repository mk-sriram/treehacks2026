# Procure

## Running the program

To run this program, first acquire the necessary API Keys for elastic, gemini, openai, elevenlabs, etc.

Then run the following commands in sequence on a UNIX-like machine
```bash
npm install
./scripts/quick-tunnel.sh # If permissions are denied, try chmod +x ./scripts/quick-tunnel.sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results
